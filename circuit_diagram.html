⚠ POWER WARNING

Each SG90 servo draws up to 700mA under load. Two servos + ESP8266 can demand 1.8A total. A standard laptop USB port only gives 500mA. Always use a powered USB hub with a 2A+ wall adapter, or you will get random resets and erratic behaviour.

⚙ SERVO PLACEMENT

Mount Servo 1 so its horn reaches the ON side of your fan switch when rotating to 55°. Mount Servo 2 so its horn reaches the OFF side when rotating to 125°. Use 3M double-sided foam tape. Fine-tune angles in the code after physical testing.

🔥 SG90 WIRE COLOURS

Red → VCC (5V)
Brown / Black → GND
Orange / Yellow → Signal (PWM)

Some clones use different colours — always verify with a multimeter if unsure.

📍 PIN SUMMARY

D1 / GPIO5 → Servo 1 signal (ON)
D2 / GPIO4 → Servo 2 signal (OFF)
VIN → 5V input (from hub)
GND → Common ground

ESP8266 is also powered via its own Micro-USB connection from the hub.
